# traste

> A library for rendering guitar and ukulele fretboards as SVG elements.

![Ukulele Fretboard](img/ukulele-fretboard.png 'UkeAlong.com Ukulele Fretboard')

## Install

```sh
npm install traste
```

## Usage

See the [examples directory](examples/).
